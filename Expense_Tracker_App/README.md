# Expense Tracker Project

This repository contains all project management documents for the Expense Tracker project.

## Folder Structure

01_Project_Charter
Project initiation documents.

02_BRD
Business Requirement Documents.

03_PRD
Product Requirement Documents.

04_Project_Plan
Schedule, WBS, milestones.

05_Risk_Register
Project risks.

06_RAID_Log
Risks, Assumptions, Issues and Dependencies.

07_Change_Requests
Approved and pending CRs.

08_Meeting_Minutes
MoMs for all meetings.

09_Status_Reports
Weekly project reports.

10_Test_Documents
Test plans and reports.

11_Final_Deliverables
Project closure documents.
